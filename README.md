# Daily English Coach 600

무료 DB 기반 영어 단어장 웹앱입니다.

## 구성
- 총 600개 단어/표현
- 해외영업 240개, 비즈니스 180개, 일상회화 180개
- 난이도: 초급 60개, 중급 420개, 고급 120개
- 최근 60일 학습 단어 제외
- 플래시카드, IPA 일부, 관련 표현, 예문, 해석
- 학습 기록, 주간 복습, 1/3/7/30일 복습, 통계

## 배포
GitHub 기존 저장소에 압축 해제한 파일 전체를 업로드하고 Commit 하면 Vercel이 자동으로 재배포합니다.

## 주의
IPA는 핵심 단어 일부에 우선 입력되어 있고, 나머지는 앱에서 '발음기호 업데이트 예정'으로 표시됩니다.
